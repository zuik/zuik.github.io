HappyTweet

Finalist project at HackBeanPot
Built with React Flask mongoDB

Ever wonder how happy you sound to others on Twitter? Or how sad? Or angry? Well, now you can use this Jackie Robinson Beanie Babies project to find out! You'll also be able to tell how other local Twitter users sound through their tweets, as well!

Stuck

Second placed project at BostonHacks
Built with Twilio API Flask

Stuck allows the user to search a word in the dictionary, find restaurants near a location, and weather in a certain area through SMS (obviously, the user does not need to be connected to the internet). Stuck can also act as a translator between two users.
